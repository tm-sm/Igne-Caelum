## Controls:


### Movement

W           -> throttle up

S           -> throttle down

A           -> pitch anti-clockwise

D           -> pitch clowckwise


### Systems

arrow keys  -> select weapons

space       -> fire weapons

E           -> use flares


### Camera

C           -> change camera mode

X           -> Select target

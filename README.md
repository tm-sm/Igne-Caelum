# Igne Caelum

2D physics-based dogfighting game created in the Godot engine. The project is abandoned for now, mostly due to university taking away most of my time.

## Controls:

### Movement

W           -> throttle up

S           -> throttle down

A           -> pitch anti-clockwise

D           -> pitch clowckwise


### Systems

arrow keys  -> select weapons

space       -> fire weapons

E           -> use flares
